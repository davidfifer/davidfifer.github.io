File Organizer

A configurable Python utility that organizes files by extension or last‑modified date, with dry‑run mode, logging, and cross‑platform file‑locking support.

Overview

The File Organizer scans a target directory and sorts files into structured subfolders based on either their file extension or their last‑modified timestamp. It supports configuration files, verbose logging, dry‑run mode, and safe handling of locked files.

Key Features

  • Extension sorting: Automatically groups files by type.
  • Date sorting: Organizes files into year/month folders.
  • Dry‑run mode: Shows actions without modifying anything.
  • Config‑driven: Supports custom file‑type mappings.
  • Logging: Detailed logs for every operation.
  • File locking: Skips locked files safely.

How It Works

The tool walks the directory tree, identifies supported file types, and moves files into destination folders based on the selected mode. It uses Python’s os, shutil, and logging modules, along with a configuration loader for custom rules.

Example Usage

python file_organizer.py --path "C:/Downloads" --mode extension --dry-run
python file_organizer.py --path "/Users/david/Documents" --mode date --verbose
