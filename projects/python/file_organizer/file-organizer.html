File Organizer

Overview

The File Organizer scans a target directory and sorts files into structured subfolders based on either their file extension or their last‑modified timestamp. It supports configuration files, verbose logging, dry‑run mode, and safe handling of locked files.

Key Features

  • Sort by Extension: Group files into subfolders based on their file type.
  • Sort by Modified Date: Organize files into date‑named folders (YYYY‑MM‑DD).
  • Dry‑run Mode: Preview all actions without modifying the filesystem.
  • Clean CLI Interface: Supports --folder_path, --sort_mode, and --verbose flags.
  • Verbose Logging: Adjustable log levels for detailed operational insight.
  • Config‑Driven Behavior: File‑type mappings and dry‑run mode defined in config.json.
  • Automatic Directory Creation: Creates destination folders on demand.
  • File‑Lock Detection: Detects locked files and retries with configurable backoff.
  • Retry Logic: Automatically retries locked or unavailable files with delay handling.
  • Graceful Error Handling: Logs warnings without interrupting the process.
  • Cross‑Platform File Lock Detection: Uses msvcrt on Windows and fcntl on Unix systems.
